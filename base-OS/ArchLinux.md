# Installation

```sh
$ loadkeys fr
$ archinstall # Follow procedure below
```

## `archinstall` procedure

---
Among abvious settings, ensure these are set to :
- Profile : Xorg & graphic drivers
- Audio : Pipewire
- Network configuration : NetworkManager
